# chef-services

```
kitchen list
kitchen converge
```
### Sets up:

1. chef-server
2. chef-analytics
3. ~~chef-delivery~~
4. chef-supermarket

### Add this to your local workstation /etc/hosts

```
33.33.33.10 chef.example.com
33.33.33.11 analytics.example.com
33.33.33.13 supermarket.example.com
```
